## Require Fast Adapter version: 5.5.1

How to use this lib in your project:
```gradle
allprojects {
	repositories {
		...
		maven { url "https://jitpack.io" }
	}
}
```

Add to your app module build.gradle
```gradle
dependencies {
	implementation "com.github.AlexDeww:FastAdapterDelegate:$last_version"
}
```
